# Weather App

A simple weather app that uses the fetch API to get weather data from openweathermap.org

## Built with

- HTML5
- SCSS (CSS Preprocessor)
- JavaScript
- Webpack
- ESLint (Airbnb styling)
- OpenWeatherMap.org one call API
